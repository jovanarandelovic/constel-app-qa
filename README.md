# constel-app-qa
Cypress automated testing project for the Social media app by Constellation.
node.js v18.13.0
Cypress v12.4.0 

## Project setup

`git clone https://github.com/jovanarandelovic/constel-app-qa.git`

`cd constel-app-qa`

`npm install`

## Run the tests

`npm run cy:open`

or

`npx cypress open`
